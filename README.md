Chat cifrado multihilo en Java con criptografía híbrida. Intercambio de clave pública RSA, cifrado de mensajes con AES y verificación de integridad con SHA-256.
> `Java` · `Sockets` · `Criptografía` · `Multithreading`

### 💬 Multithreaded Socket Chat Server
Servidor de chat en Java que gestiona múltiples clientes de forma concurrente usando hilos. Comunicación en tiempo real vía sockets TCP.
> `Java` · `Sockets` · `Concurrencia`

Servidor web implementado desde cero en Java con soporte para peticiones HTTP y HTTPS. Manejo de rutas, cabeceras y respuestas estáticas.
> `Java` · `HTTP` · `SSL/TLS`
